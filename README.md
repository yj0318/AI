# 🚗 도로 표지판 구분 프로젝트

## 🚌 프로젝트 소개와 목적

* YOLOv5를 이용하여 사진을 훈련하고 검증하여 임의의 데이터를 입력 받았을 때 그 데이터를 적절한 클래스에 분류하는 프로그램을 구현합니다.
